# Fruit Image Classification

## Project Overview

This project focuses on fruit image classification using deep learning methods. The goal is to classify fruit images into four categories:

- Apple
- Banana
- Orange
- Mixed fruits

The project starts from a baseline CNN model and gradually improves performance through architecture experiments, input resolution studies, regularization, class balancing, data augmentation, model optimization, and transfer learning.

## Methods

The project includes the following experiments:

Dataset analysis and class distribution checking
Baseline CNN model training
CNN architecture depth comparison
Input image resolution comparison
Regularization experiments using Dropout and Batch Normalization
Class balance and data augmentation experiments
Model optimization and training strategy improvement
Transfer learning using pretrained deep learning models
Main Techniques
Python
Jupyter Notebook
Convolutional Neural Networks
Data augmentation
Transfer learning
Model evaluation
Confusion matrix analysis
Results Summary

The project compares model performance under different experimental settings, including CNN depth, image resolution, regularization, class balancing, augmentation, and transfer learning.

The final transfer learning experiments achieved the best classification performance among all tested approaches.

The full image dataset is not included in this repository due to file size considerations. The notebooks are provided for project demonstration, experiment documentation, and portfolio review.
